# Sales Prediction Using Machine Learning

## Problem Statement
This project focuses on predicting product sales using advertising expenditure data. Sales prediction helps businesses plan marketing strategies and make informed decisions.

## Dataset
A public Advertising dataset was used for this project. The dataset contains advertising budgets spent on TV radio and newspaper platforms along with sales values.
https://raw.githubusercontent.com/justmarkham/scikit-learn-videos/master/data/Advertising.csv

## Model Used
Linear Regression was used because it is simple and easy to interpret. It helps explain the relationship between advertising expenditure and sales.

## Evaluation
The model was evaluated using Mean Absolute Error Root Mean Squared Error and R squared score.

## Conclusion
The model was able to predict sales with reasonable accuracy and the results align with real world marketing trends.
